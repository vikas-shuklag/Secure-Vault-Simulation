# Contributing to Virtual HSM

We love pull requests! If you want to contribute to this project:
1. Fork the repository
2. Create a feature branch (`git checkout -b feature/AmazingUpdate`)
3. Commit your changes
4. Push to the branch
5. Open a Pull Request

## Development Setup
- Install dependencies via `pip install -r requirements.txt`
- Install React via `cd frontend && npm install`
- Start the API: `python -m uvicorn api.main:app`
- Start the frontend: `npm run dev`
